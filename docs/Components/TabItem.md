# Tab Item

More information available at: [NavigationBar]("https://shunnedreality.github.io/material-roblox/components/tabs")

## API
```typescript
TabItem(
	scope: Fusion.Scope,
	props: {
		label: string,
		icon: string
	}
)
```

## Usage
```lua
local MaterialRoblox = require(MaterialRoblox);

MaterialRoblox.Components.TabItem(scope, {
    label = "Video",
    icon = "video"
}),
```
